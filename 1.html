HTML

Hypertext Markup Language (HTML) is the standard Markup language for creatig web pagesand web applications.
It is the dominant markup language for web pages in the sense of herbal marking language and hypertext marking language.
HTML provides a way to create structured documents out of
links, citations, and other items, as well as presenting structural meaning for text such as
headings, paragraphs, lists, and so on. It can also be used to embed images and objects and create interactive forms.

HTML is written in the form of an HTML element that is a "tag" surrounded by angle brackets within the web page content.
HTML can contain or load scripts such as JavaScript that affect the behavior of
HTML processing devices, such as web browsers, and CSS that define the appearance and placement of text and other items.
W3C, co-director of the HTML and CSS standards, encourages the use of CSS for clear and representative markup.
